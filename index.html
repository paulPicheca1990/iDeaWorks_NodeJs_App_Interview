Here you will discover my wicked application I developed over the weekend with NodeJs.

What does this appplication do?

This application will act as an API as well as an HTML/HTTP frontend for users. This application will track users first-name, middle-name, last-name, gender, date of birth, address, ontario health card number, and any other generic notes.

What was used for the Database portion?

This application relied on MongoDB, along with the mongoose package for NodeJs. Together, they welded and formed the data which made my life a hell of a lot easier.

What framework was used within this NodeJs application?

This application used ExpressJs for the framework, and EJS for the templating of the server-pages.
